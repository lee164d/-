# StarEyeCat — AI-Driven Content Pipeline

Multi-agent system for automated short-video content creation and digital asset production.

## Architecture
```
Scout (data gathering) → Brain (script writing) → Forge (asset creation) → Analyst (performance review)
```

## Setup
1. `cp config/.env.example .env` and fill in your API keys
2. `npm install`
3. `npm run build`
4. `npm test`
